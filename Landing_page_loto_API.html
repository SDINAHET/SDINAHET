Loto Tracker API

A powerful API that tracks Loto results, compares tickets, and manages user accounts securely.

Features

Secure Authentication

JWT-based authentication ensuring user security.

Hybrid Database

Supports both SQL (MySQL, SQLite) and NoSQL (MongoDB).

Results Analysis

Automatically compares user tickets with official results.

About

The **Loto Tracker API** is a **portfolio project** for Holberton School Rennes, aimed at automating Loto result tracking and ticket analysis. It leverages **Spring Boot** for backend development, **JWT authentication** for security, and supports **SQL (MySQL, SQLite)** and **NoSQL (MongoDB)** for flexible storage solutions.

This project provides an easy way for users to check Loto results, compare their tickets automatically, and keep track of their earnings over time.
